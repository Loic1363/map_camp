# MapCamp

**MapCamp** est une application **desktop Python** développée avec **Tkinter** et **TkinterMapView**, permettant de **visualiser, rechercher, ajouter, modifier et gérer des lieux de campings** sur une carte interactive basée sur **OpenStreetMap**.

---

## Fonctionnalités

- **Affichage interactif d’une carte** avec **zoom** et **position personnalisables**.  
- **Recherche géographique** des lieux (adresse, ville, rue) via **OpenStreetMap (Nominatim)**.  
- **Ajout de nouveaux campings** directement sur la carte, avec **nom**, **date**, et **commentaires**.  
- **Liste des lieux enregistrés** avec possibilité de **modification** et **suppression**.  
- **Export / import** des points au format **JSON** pour un **partage facile entre machines**.  
- **Interface optimisée** avec **barre de recherche stylisée** comme **Google Maps**.
