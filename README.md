# MapCamp

**MapCamp** is a **desktop Python application** developed with **Tkinter** and **TkinterMapView**, allowing you to **visualize, search, add, edit, and manage camping locations** on an interactive map based on **OpenStreetMap**.

---

## Features

- **Interactive map display** with **customizable zoom** and **position**.  
- **Geographical search** for places (address, city, street) via **OpenStreetMap (Nominatim)**.  
- **Add new campsites** directly on the map, with **name**, **date**, and **comments**.  
- **List of saved locations** with options to **edit** and **delete**.  
- **Export / import** points in **JSON format** for **easy sharing between devices**.  
- **Optimized interface** with a **Google Mapsâ€“style search bar**.
